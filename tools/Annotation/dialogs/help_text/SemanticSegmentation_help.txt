Create a new semantic segmentation datastream: 
	Tools -> Add semantic segmentation datastream
	Set a name and a new filter ID
	Overlay filter ID must be an existing monocamera filter ID
	
Draw a new shape:
	Press `Insert` or `Draw shape` button
	Add points with left-click
	Press `Insert` or `Stop Drawing shape` button

Modify a shape:
	Select the shape
	Select another class if needed
	You can drag the points with the mouse or move it with the arrows
	
Delete a shape:
	Select a shape
	Press `Delete` or `Delete Shape` button
	
Save changes:
	Press `Save` button or `Ctrl+S`

Copy  - Ctrl+C
Cut   - Ctrl+X
Paste - Ctrl+V
